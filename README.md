# RISK-Game Java Simulation Engine & Web Visualizer

A Java-based simulation engine for the strategy board game Risk, plus a lightweight web visualizer (HTML/CSS/JS) to explore simulation results and player performance.

## Live Demo
Web Visualizer: https://abdelrahmanalashel.github.io/RISK-Game/

---

## What this project does
- Simulates Risk gameplay using an object-oriented Java engine (game state, players, territories, etc.).
- Runs automated games using multiple strategies.
- Collects results across repeated trials (wins/losses, turns, attacks/defenses, territories gained/lost, etc.).
- Visualizes outcomes in a browser dashboard for quick comparison.

---

## Repo Structure
- `src/main/java/` — Java implementation
- `src/test/java/` — JUnit tests
- `web/` — web visualizer (deployed via GitHub Pages)
- `assets/screenshots/` — photos used in thE REPORT
- `REPORT.md` — experiment write-up and findings
